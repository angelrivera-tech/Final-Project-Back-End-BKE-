# Final-Project-Back-End-BKE-
This Project is a basic back end to a application of task manager.

To make it work first download the source code that is in the zip file. After you do so unzip the file and open said file in vs code (visual studio code). After that download ssms if you dont have it and paste the database code in the initsql folder into a new query in ssms. press execute then go back to vs code open a new terminal and add "node src/app.js" and press enter. then you should be able to go to post man paste the link the server gave you in the vs terminal and start trying out the CRUD operations.
